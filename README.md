# Applied Math & Algorithms in C

*🌍 [English](#english) | 🇷🇺 [Русский](#russian)*

---

<h2 id="english">🇬🇧 English</h2>

*   **Riccati Equation Solver:** A C program for numerically solving the Continuous-time Algebraic Riccati Equation (CARE). Implements a two-tier iterative algorithm using the Newton-Kleinman method and the Conjugate Gradient Normal Equation (CGNE) method for Lyapunov equations.

---

<h2 id="russian">🇷🇺 Русский</h2>

*   **Решатель уравнения Риккати:** Программа на C для численного решения обобщенного алгебраического уравнения Риккати (CARE). Реализован двухуровневый итерационный алгоритм: метод Ньютона-Клейнмана + метод сопряженных градиентов на нормальных уравнениях (CGNE).
